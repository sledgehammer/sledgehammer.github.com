Example

where('apple') returns the items with the value is "apple" (1D array) where('> 5') returns the items where the value is greater-than 5 where(array('id' => 4)) returns the items where the element or property 'id' is 4 where(array('user->id <' => 4)) returns the items where the property 'id' of element/property 'user' is smaller-than 4 where(function ($item) { return (strpos($item, 'needle') !== false); }) return the items which contain the text 'needle'
