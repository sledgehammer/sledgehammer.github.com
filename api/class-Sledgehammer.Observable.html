Class Observable

Observable, an Event/Listener

Example subclass: class Button extends Observable {
protected $events = array( 'click' => array() ); // optional method that gets executed when the 'click' event is fired. protected function onClick($sender) {} }

$button = new Button(); $button->on('click', function ($sender) { echo 'clicked'; });

// Or property style event-binding: (Only 1 listeners per event)

$button->onClick = function () { echo 'clicked'; };

// reset all listeners. $this->onClick = null;
