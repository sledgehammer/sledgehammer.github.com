Function getClientIp

Vraagt het IP adres van de client op.

Deze methode biedt MINDER beveiling dan de $SERVER['REMOTE_ADDR'] en is makkelijk te spoofen, Maar in een (reverse) proxy of load-balanced situatie geeft de $_SERVER['REMOTE_ADDR'] het de IP van de load-balancer. En ben je dus afhankelijk van de HTTP vars.
