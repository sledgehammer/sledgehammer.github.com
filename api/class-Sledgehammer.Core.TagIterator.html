Class TagIterator

Een iterator die de tags uit htmlcode haalt. Met name geschikt voor html met fouten, zoals bijvoorbeeld een "header.php" waarbij een aantal tags niet worden gesloten. Met de uitvoer kun je de exacte (foute) html weer reconstrueren.

Vaak is een oplossing mogelijk met DOMDocument of SimpleXML. Gebruik in die gevallen niet deze iterator.
