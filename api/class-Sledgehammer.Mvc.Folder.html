Class Folder

Superclasse van de Virtuele mappen. Door Folder creer je eenvoudig virtuele mappen en virtuele bestanden. Hierdoor heb je vrijheid in de paden die je gebruikt om de pagina's aan te duiden. I.p.v. "page.php?id=17" maak je "pages/introductie.html" tevens kun je viruele mappen nesten (Een virtuele map in een virtuele map) hierdoor kan een een hele map hergebuiken en parameterizeren.

protected integer $depthIncrement

Het aantal niveau's(submappen) dat door deze VirtualFolder wordt afgehandeld. Deze variabele wordt gebruikt om de $depth van de submap uit te rekenen. Dit is handig als je een andere VirtualFolder wilt gebruiken terwijl je zelf al meerdere submappen gebruikt. Als je je de $depth_increment op 0 zet, dan wordt de andere VirtualFolder niet als subfolder,maar als de dezelfde folder gebruikt.
