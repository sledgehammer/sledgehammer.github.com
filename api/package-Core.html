Package Core

Classes summary
Sledgehammer\AutoLoader Load class and interface definitions on demand. Improves performance (parsetime & memory usage), only classes that are used are loaded.
Sledgehammer\Collection Collection: Array on Steriods Provides a filtering, sorting, events and other utility functions for collections.
Sledgehammer\CSV Een generieke snelle manier om een csv bestand in te lezen. Kan ook csv bestanden genereren met CSV::write()
Sledgehammer\cURL cURL, an HTTP/FTP response object Simplifies asynchronous requests & paralell downloads. Uses the curl functions and throws exceptions on errors.
Sledgehammer\Database A PDO Database class with additional debugging functions. By default will report clean sql-errors as notices, sets the encoding to UTF8 and sets the default fetch behaviour to FECTH_ASSOC
Sledgehammer\DatabaseCollection DatabaseCollection a Collection interface to a database result. It will lazyily generate & mutate the SQL query based on the filter & sorting operations. Inspired by "Linq to SQL"
Sledgehammer\DatabaseSessionHandler Sessie gegevens 'centraal' opslaan in een database Hierdoor kan een Loadbalancer in 'round robin' mode draaien.
Sledgehammer\Dump Parses a var_dump() and renders a syntax highlighted version of var_export()
Sledgehammer\ErrorHandler Verzorgt de afhandeling van php fouten. Deze kan een uitgebreide foutmelding in html-formaat tonen en/of emailen
Sledgehammer\Framework Container voor Sledgehammer Framework functions - Module detection and initialisation - Language & locale initialisation
Sledgehammer\HTML HTML, an view compatible class for rendering raw html. Contains static helper functions for generating tags, etc.
Sledgehammer\HTMLTokenizer Een tokenizer voor htmlcode. Met name geschikt voor html met fouten. Met de uitvoer kun je de exacte (foute) html weer reconstrueren.
Sledgehammer\HttpAuthentication Basic HTTP autentication
Sledgehammer\Json Renders the data as Json Compatible with MVC's the Document/View inferface.
Sledgehammer\Object A generic php superclass.
Sledgehammer\Observable Observable, an Event/Listener
Sledgehammer\PDOStatement PDOStatement override Adds Countable to the Database results.
Sledgehammer\PHPAnalyzer Statically Analyzes PHP code and collects data about class and interface usage and deflarations.
Sledgehammer\PHPTokenizer A tokenizer that gives context to tokens of php internal tokenizer. Serves as a helper for the PHPAnalyer class.
Sledgehammer\PreparedStatement A PDOStatment subclass that logs prepared statements. Logs the execution time and executed query to the connected Database.
Sledgehammer\PropertyPath An helper class that resolves properties inside arrays and objects based on a path. Inspired by XPath But implemented similar to "Property Path Syntax" in Silverlight
Sledgehammer\Readonly A wrapper that prevents changes to an object or array.
Sledgehammer\SQL Een object waar een complexe SQL query mee kunt samenstellen & bewerken. Has a fluent interface where the methods won't modify the object but will return a new object.
Sledgehammer\Stopwatch Helper class for manual performance profiling.
Sledgehammer\TagIterator Een iterator die de tags uit htmlcode haalt. Met name geschikt voor html met fouten, zoals bijvoorbeeld een "header.php" waarbij een aantal tags niet worden gesloten. Met de uitvoer kun je de exacte (foute) html weer reconstrueren.
Sledgehammer\Text Text, a string class for handeling (multibyte) strings with OOP syntax Modelled after the C# String class.
Sledgehammer\URL Utility class for generating and manipulating urls.
Sledgehammer\Wrapper Wrap the object/array into container object Allow filters and accesscontrol to any object or array.
Exceptions summary
Sledgehammer\InfoException An exception with additonal information for the ErrorHander.
Functions summary

deprecated Report deprecated functionality.
dump Dumps information about a variable, like var_dump() but with improved syntax and coloring.
error Report a fatal error (and stop executing).
notice Report a notice
report_exception Report an exception
Sledgehammer\array_key_unshift Prepend/moves the value to the beginning of the array using the specified $key.
Sledgehammer\browser Retrieve browser and OS info
Sledgehammer\call_static_func Call a static method on a specific class without Late Static Binding. Using "call_user_func" will bind the get_called_class() to subclass when the method is inside the parent class.
Sledgehammer\collection Shorthand for creating an Collection object Allows direct method-chaining: collection($array)->select('name');
Sledgehammer\compare Compare two values with the given operator. For '==' the values are compared with strcasecmp().
Sledgehammer\copydir Een map inclusief bestanden en submappen copieren (Als de doelmap al bestaat, worden deze overschreven/aangevuld) De doelmap wordt NIET eerst verwijderd
Sledgehammer\equal_properties Vergelijk de eigenschappen van 2 objecten met de equals functie.
Sledgehammer\equals Controleerd of 2 variabelen gelijk of bijna gelijk aan elkaar zijn. equals((float) 1.000, (int) 1) == true equals("1.1", 1.1) == true equals("abc", "ABC") == f
Sledgehammer\extend_include_path Prepends the $path to the include path
Sledgehammer\extend_time_limit Hiermee kun je de timeout van het script verlengen. Bij $relative op true zal de timeout aanvuld worden zodat het script nog minimaal n seconden mag draaien. (De reeds ingestelde timeout zal nooit ingekort worden) Bij $relative op false (absolute) zullen de $seconden bij de huidige timeout worden opgeteld.
Sledgehammer\extract_element Een element uit een array opvragen. Kan ook een subelement opvragen "element[subelement1][subelement2]" Retourneert true als de waarde is gevonden. [abc] wordt ['abc'], maar [1] wordt geen ['1']
Sledgehammer\file_extension Deze functie geeft de extentie van de bestandnaam terug.
Sledgehammer\format_parsetime Zet een float om naar een leesbare (parse)tijdnotatie 61.23 => '1:01.230'
Sledgehammer\get_public_methods Werkt net als get_class_methods, maar de array bevat alleen de publieke functies
Sledgehammer\get_public_vars Get the properties that are publicly accessable
Sledgehammer\getClientIp Vraagt het IP adres van de client op.
Sledgehammer\getDatabase Get the database instance. Will autoconnect to the database configured in "/application/settings/database.ini"
Sledgehammer\human_implode implode(), maar dan zodat deze leesbaar is voor mensen. Bv: echo human_implode(' of ', array('appel', 'peer', 'banaan')); wordt 'appel, peer of banaan'
Sledgehammer\is_assoc Test of een array een assoc array is. (Als een key NIET van het type integer zijn)
Sledgehammer\is_indexed Test of een array een indexed array is. (Als de keys liniair zijn opgebouwd)
Sledgehammer\iterator_keys Geeft de keys van een iterator in een array. Net als array_key(), maar dan voor iterators.
Sledgehammer\iterators_to_arrays Zet alle iterators binnen $data om naar arrays. Doet zijn best om $source niet te veranderen, maar is afhankelijk van __clone (private en protected eigenschappen worden niet omgezet)
Sledgehammer\mimetype Detect mimetype based on file-extention
Sledgehammer\mkdirs Het path creeren Zal alle map-namen die in het path genoemd worden proberen te maken. Zal geen foutmelding geven als het path al bestaat.
Sledgehammer\mtime_folders Get the timestamp for the latest change in the directory.
Sledgehammer\partial_var_export Returns human-readable information about a object or array variable. Used in syntax_hightlight() title (Shows the contents of an object/array on hover)
Sledgehammer\quoted_human_implode human_implode(), but with quotes (") around the values.
Sledgehammer\quoted_implode implode(), but with quotes (") around the values.
Sledgehammer\redirect Een redirect naar een andere pagina. Werkt indien mogelijk via de HTTP header en anders via Javascript of een META refresh tag.
Sledgehammer\render_file Sends the contents of the file including appropriate headers. Zal na het succesvol versturen van het bestand het script stoppen "exit()"
Sledgehammer\rmdir_contents Delete the contents of the folder, but not the folder itself.
Sledgehammer\rmdir_recursive De map, inclusief alle bestanden en submappen in het $path verwijderen.
Sledgehammer\safe_unlink Een bestand verwijderen, met extra controle dat het bestand zich in de $basepath map bevind.
Sledgehammer\select Shorthand for creating an SQL object and selecting columns. Allows direct method-chaining: select('*')->from('table')->where('column = "value"');
Sledgehammer\sem_key Genereer aan de hand van de $identifier een (meestal) uniek id
Sledgehammer\send_headers De HTTP headers versturen.
Sledgehammer\set_object_vars Werkt als get_object_vars() maar i.p.v. de waardes op te vragen worden deze ingesteld
Sledgehammer\statusbar Show debug and profiling information. Contains parsetime, memory usage and (sql)querylogs.
Sledgehammer\su_exec Return the output of a shell command that is run as another user.
Sledgehammer\sudo Execute a shell command as another user.
Sledgehammer\syntax_highlight Returns a html snippet with the variable in a predefined color per type. Used in Dump & ErrorHandler
Sledgehammer\text Shorthand for creating an Text object Allows direct method-chaining: text('my text')->toUpper()->trim();
Sledgehammer\urlencode_path Mappen en bestandsnaam in de url corrigeren Geen " " maar "%20" enz
Sledgehammer\wget Bestand downloaden(naar het geheugen) en opslaan. retourneert de groote van het bestand (in karakters)
Sledgehammer\write_ini_file Write an ini file, compatible with parse_ini_file()
