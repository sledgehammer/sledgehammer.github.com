Classes summary

Sledgehammer\ActiveRecord An ActiveRecord frontend for the Repository
Sledgehammer\ArrayRepositoryBackend A RepositoryBackend for 1 model stored in an array.
Sledgehammer\BelongsToPlaceholder This Placeholder facilitates lazy loading of belongsTo relations. A BelongsToPlaceholder object behaves like the object from the repository, but only retrieves the real object on-access or on-change.
Sledgehammer\CollectionView Extract the key/value of the collection based on a propertypath.
Sledgehammer\DatabaseRepositoryBackend Repository backend for database records
Sledgehammer\HasManyPlaceholder This Placeholder facilitates lazy loading of hasMany relations. A HasManyPlaceholder object behaves like an Collection containing all related objects from the repository, but only retrieves the objects on-access or on-change.
Sledgehammer\Inflector The Inflector transforms words from singular to plural and from plural to singular
Sledgehammer\ModelConfig Model configuration class, a formal definition of a Repository model.
Sledgehammer\Repository Repository/DataMapper
Sledgehammer\RepositoryBackend The minimal interface for a (read-only) Repository Backend
Sledgehammer\RepositoryCollection A Collection containing repository instances. Contains the raw \Traversable|array from the backend and converts the data to instances on-access.
Sledgehammer\SimpleRecord An ActiveRecord with (generated) properties based on the ModelConfig->properties. Adds the ActiveRecord interface to any Repository model, but disables support for property-autocompletion.
