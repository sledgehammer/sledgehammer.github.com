cache Returns the cached value when valid cache entry was found. otherwise retrieves the value via the $closure, stores it in the cache and returns it.
call_static_func

Call a static method on a specific class without Late Static Binding. Using "call_user_func" will bind the get_called_class() to subclass when the method is inside the parent class.
collection

Shorthand for creating an Collection object Allows direct method-chaining: collection($array)->select('name');.
compare

Compare two values with the given operator. For '==' the values are compared with equals().
component_to_string

Geeft de uitvoer van een component als string. (Uitvoer zoals emails en header() worden niet afgevangen).
copydir

Een map inclusief bestanden en submappen copieren (Als de doelmap al bestaat, worden deze overschreven/aangevuld) De doelmap wordt NIET eerst verwijderd.
debugr

Shorthand for sending DebugR messages. debugr()->log($var); instead of \Sledgehammer\DebugR::log($var); debugr($var); instead of \Sledgehammer\DebugR::dump($var).
deprecated Report deprecated functionality.
dump Dumps information about a variable, like var_dump() but with improved syntax and coloring.
equal_properties Vergelijk de eigenschappen van 2 objecten met de equals functie.
equals Check if two variables are equal with minimal type-coercion.
error Report a fatal error (and stop executing).
explode_xml_parameters

Zet een string met parameters om naar een array. ' x="y"' wordt array('x' => 'y').
extend_include_path Prepends the $path to the include path.
extend_time_limit

Hiermee kun je de timeout van het script verlengen. Bij $relative op true zal de timeout aanvuld worden zodat het script nog minimaal n seconden mag draaien. (De reeds ingestelde timeout zal nooit ingekort worden) Bij $relative op false (absolute) zullen de $seconden bij de huidige timeout worden opgeteld.
extract_element

Een element uit een array opvragen. Kan ook een subelement opvragen "element[subelement1][subelement2]" Retourneert true als de waarde is gevonden. [abc] wordt ['abc'], maar [1] wordt geen ['1'].
extract_logical_operator Extract and return the logical operator of a $conditions array.
file_extension Deze functie geeft de extentie van de bestandnaam terug.
filter Filter a variable using a callable.
format_parsetime

Zet een float om naar een leesbare (parse)tijdnotatie 61.23 => '1:01.230'.
get_public_methods Werkt net als get_class_methods, maar de array bevat alleen de publieke functies.
get_public_vars Get the properties that are publicly accessable.
getClientIp Vraagt het IP adres van de client op.
getDatabase
getRepository

Get a Repository by ID This allows instances to reference a Repository by id instead of a full php reference. Keeping the (var_)dump clean.
human_implode

implode(), maar dan zodat deze leesbaar is voor mensen. Bv: echo human_implode(' of ', array('appel', 'peer', 'banaan')); wordt 'appel, peer of banaan'.
image Shorthand for creating an Image object.
imagecreatefrombmp Create a GD resource from a BMP file.
implode_xml_parameters Zet een array om naar xml/html parameters; array('x' => 'y') wordt ' x="y"'.

is_closure Checks if a variable is a closure.
is_indexed Test of een array een indexed array is. (Als de keys liniair zijn opgebouwd).
is_valid Validate a variable using a callable.
is_valid_component Check if $component is compatible with the Component interface, otherwise report notices.
iterator_keys Geeft de keys van een iterator in een array. Net als array_key(), maar dan voor iterators.
iterators_to_arrays

Zet alle iterators binnen $data om naar arrays. Doet zijn best om $source niet te veranderen, maar is afhankelijk van __clone (private en protected eigenschappen worden niet omgezet).
merge_headers

Merge the results from the $component->getHeaders(). The headers from the $component overwrites the values in $headers.
mimetype Detect mimetype based on file-extention.
mkdirs

Het path creeren Zal alle map-namen die in het path genoemd worden proberen te maken. Zal geen foutmelding geven als het path al bestaat.

notice Report a notice.
once

Creates a version of the function that can only be called one time. Repeated calls to the returned closure will have no effect, returning the value from the original call.
partial_var_export

Returns human-readable information about a object or array variable. Used in syntax_hightlight() title (Shows the contents of an object/array on hover).
quoted_human_implode human_implode(), but with quotes (") around the values.
quoted_implode implode(), but with quotes (") around the values.
redirect

Een redirect naar een andere pagina. Werkt indien mogelijk via de HTTP header en anders via Javascript of een META refresh tag.
reflect_properties Use Reflection to extract all properties.
render_file

Sends the contents of the file including appropriate headers. Zal na het succesvol versturen van het bestand het script stoppen "exit()".
report_exception Report an exception.
rmdir_contents Delete the contents of the folder, but not the folder itself.
rmdir_recursive De map, inclusief alle bestanden en submappen in het $path verwijderen.
safe_unlink Een bestand verwijderen, met extra controle dat het bestand zich in de $basepath map bevind.
select

Shorthand for creating an SQL object and selecting columns. Allows direct method-chaining: select('*')->from('table')->where('column = "value"');.
sem_key Genereer aan de hand van de $identifier een (meestal) uniek id.
send_headers De HTTP headers versturen.
set_object_vars Werkt als get_object_vars() maar i.p.v. de waardes op te vragen worden deze ingesteld.
statusbar

Show debug and profiling information. Contains parsetime, memory usage and (sql) log entries.
su_exec Return the output of a shell command that is run as another user.
sudo Execute a shell command as another user.
syntax_highlight

Returns a html snippet with the variable in a predefined color per type. Used in Dump & ErrorHandler.
text

Shorthand for creating an Text object Allows direct method-chaining: text('my text')->toUpper()->trim();.
tidy_id ID and NAME must begin with a letter ([A-Za-z]) and may be followed by any number of letters, digits ([0-9]), hyphens ("-"), underscores ("_"), colons (":"), and periods (".").
urlencode_path

Mappen en bestandsnaam in de url corrigeren Geen " " maar "%20" enz.
value

Return the value of a variable or return null if the valiable not exist. (Prevents "Undefined variable" notices) WARNING! As a side-effect non existing variables are set to null. If you pass array element to value($var['index']) and $var didn't exist, an array is created: array('index' => null) Use \Sledgehammer\array_value() which doesn't have this side effect for array.
