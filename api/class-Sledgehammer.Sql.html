Class Sql

Query object to mutate and generate (complex) SQL queries.

Sledgehammer\Object
Sledgehammer\Sql
Namespace: Sledgehammer
Package: Core
Link: Doesn't have a "fluent" interface: Methods won't modify the object (and return $this) but will return a new object. $sql = select('*')->from('customers'); // uses the select shorthand $sql2 = $sql->where('id = 1'); echo $sql; // "SELECT * FROM customers" echo $sql2; // "SELECT * FROM customers WHERE id = 1" More on method chaining: http://stackoverflow.com/questions/1103985/method-chaining-why-is-it-a-good-practice-or-not#7215149
